# Tutorial Joga da Velha React

Este repositório é a implementação do jogo da velha (tic tac toe) seguindo o tutorial React.

## Requisitos
* Node.js
* npm
* npx

## Preparação de ambiente
```bash
    npm install
    npx create-react-app <nome do app>
```

## Inicialização da aplicação
```bash
    npm start
```
